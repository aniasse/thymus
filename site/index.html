Thymus

Le système immunitaire de votre réseau.

Voir le code Découvrir l'approche
Les solutions classiques cherchent les attaques connues. Thymus fait l'inverse : il apprend l'identité comportementale du réseau — son « Soi » — et réagit à toute mutation impossible. Exactement comme le système immunitaire humain ne connaît pas toutes les maladies, mais connaît intimement les cellules du corps.
L'approche

Un réseau est un organisme vivant

Chaque machine possède un ADN comportemental : avec qui elle parle, quand, et combien. Quand ce comportement mute brutalement, c'est le signe d'une compromission — même sans malware, même avec des identifiants valides.

Fonctionnement

Léger, souverain, déployable

Des capteurs observent, un cœur en Rust analyse, un dashboard montre tout en temps réel. Mode agent (Linux, Windows/ETW) ou mode passif sur port miroir — y compris les appareils sans agent (imprimantes, caméras IP, IoT).

01 — OBSERVER

Capteurs

Collectent connexions, processus et volumes. Agent ou capture passive (SPAN).

02 — APPRENDRE

Cœur immunitaire

Construit l'ADN du réseau, détecte les mutations et les déplacements latéraux.

03 — RÉAGIR

Réponse graduée

Surveillance, limitation, isolement — et alertes webhook.

04 — VISUALISER

Dashboard

Cartographie vivante, mutations, types d'appareils, chaînes d'attaque.
